## Install

- **macOS** — Open the `.dmg` file and drag Coodeen to your Applications folder.
- **Windows** — Run the `.exe` installer and follow the prompts.
- **Linux** — Make the `.AppImage` executable (`chmod +x Coodeen-linux.AppImage`) and run it.

## Configure a Provider

Launch Coodeen and click the **gear icon** in the top bar to open Settings. Add an API key for at least one provider:

| Provider  | Get API Key                                                       | Models                                  |
| --------- | ----------------------------------------------------------------- | --------------------------------------- |
| OpenAI    | [platform.openai.com](https://platform.openai.com/api-keys)      | GPT-4o, GPT-4.1, GPT-4.1-mini          |
| Anthropic | [console.anthropic.com](https://console.anthropic.com/)           | Claude Sonnet 4, Claude Haiku           |
| Google    | [aistudio.google.com](https://aistudio.google.com/apikey)        | Gemini 2.5 Pro, Gemini 2.5 Flash       |

## Example Workflow

1. Launch Coodeen
2. Select your project directory using the folder picker in the top bar
3. Add your API key in Settings
4. Point the preview panel to your dev server (e.g. `http://localhost:3000`)
5. Start chatting — ask the AI to build features, fix bugs, or explain code
6. Capture screenshots from the preview to give the AI visual context
