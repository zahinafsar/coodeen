## Prerequisites

- [Bun](https://bun.sh/)
- [Node.js 18+](https://nodejs.org/)
- Git

## Setup

```bash
git clone https://github.com/zahinafsar/coodeen.git
cd coodeen
bun install
```

## Running Locally

```bash
# Start the desktop app in development mode
bun run dev
```

This launches the Electron app with hot reload — changes to the renderer update instantly.

## Type Checking

```bash
bun run --cwd apps/desktop typecheck
```

## Building

```bash
# Build the app for production
bun run build

# Build and package as an installer for your platform
cd apps/desktop && bun run build:prod
```

## Project Structure

| Directory        | Purpose                                |
| ---------------- | -------------------------------------- |
| `apps/desktop`   | Electron desktop app (main + renderer) |
| `apps/docs`      | Documentation site (Next.js + Fumadocs)|
| `packages/cli`   | Legacy CLI package                     |
| `scripts/`       | Build helper scripts                   |
