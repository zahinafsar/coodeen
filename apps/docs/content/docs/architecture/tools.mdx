## Overview

When you chat with the AI, it can take actions on your codebase automatically. You'll see each action displayed inline in the conversation so you always know what's happening.

## What the AI Can Do

### Read and Search

- Read files in your project
- Search for files by name or pattern
- Search inside files for specific text
- List folder contents
- Fetch content from URLs

### Write and Edit

- Create new files
- Edit existing files
- Apply changes across multiple files
- Save images to disk

### Run Commands

- Execute shell commands in your project directory (install packages, run tests, start servers, etc.)

### Plan

- Write and manage plans before making changes
- Create todo lists to track progress
- Ask you questions to clarify requirements

### Other

- Run registered skills
- Fetch images from the web for reference

## How It Works

The AI decides which actions to take based on your request. Every action and its result is shown in the chat panel — you can expand each one to see the details. In **Plan mode**, the AI can only read your code and propose changes without modifying anything.
