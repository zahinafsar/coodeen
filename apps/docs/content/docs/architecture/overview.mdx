## Overview

Coodeen is a native desktop app built with Electron. Everything runs locally on your machine — there's no server to set up or manage.

## What's Inside

- **AI chat** — Powered by the Vercel AI SDK, supporting OpenAI, Anthropic, and Google models
- **File access** — The AI can read, write, and search files in your project
- **Terminal** — Built-in shell powered by node-pty
- **Git** — View branches, status, and commit history
- **Database** — SQLite stores your sessions, messages, and settings locally
- **Preview** — Embedded browser frame showing your running app

## Data Storage

All your data stays on your machine:

| Platform | Location |
|----------|----------|
| macOS | `~/Library/Application Support/coodeen/` |
| Windows | `%APPDATA%/coodeen/` |
| Linux | `~/.config/coodeen/` |

## Open Source

Coodeen is fully open source. The code is on [GitHub](https://github.com/zahinafsar/coodeen) and contributions are welcome.
