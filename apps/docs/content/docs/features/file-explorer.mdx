## Overview

The file explorer panel provides a tree view of your project directory. You can browse files, view their contents, and manage your project structure.

## Features

- **Directory tree** — Navigate your project with an expandable folder structure
- **File viewing** — Click any file to view its contents with syntax highlighting
- **File creation** — Create new files and directories from the explorer
- **File deletion** — Remove files you no longer need

## Usage

Switch to the file explorer tab in the right panel to see your project files. The tree automatically reflects the project directory configured in the top bar.

The AI agent also has full access to your filesystem through its tools — you can ask it to read, create, edit, or delete files directly from the chat.
