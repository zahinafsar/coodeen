## Overview

The terminal toggle in the preview toolbar opens a shell pane below the preview. It uses [`xterm.js`](https://xtermjs.org/) (with the fit and web-links addons) in the renderer, wired to a real PTY in the main process via [`@lydell/node-pty`](https://github.com/lydell/node-pty).

## Details

- Shell is `$SHELL` or `/bin/sh`, spawned in the session's project directory.
- Terminal is xterm-256color with fit-to-panel resizing.
- Processes are tracked in a session map and killed cleanly on window close.
- The IPC surface (`pty:spawn`, `pty:write`, `pty:resize`, `pty:kill`, `pty:list`) is exposed in `handlers/pty.ts`.

## Agent Access

Opencode's `bash` tool can run shell commands directly; the integrated terminal is for when you want to type them yourself.
